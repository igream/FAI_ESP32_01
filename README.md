# FAI_ESP32_01

Nota: Esto es simplemente una emulación de envio de datos a render, simula una senoidal a través de 24 horas,
modifica los valores de la senoidal y la zona horaria que corresponda.

Utiliza tus API Keys de MongoDB, este programa fue hecho antes de Actions, se necesitaba un bot consultador para que la página no cayera.

Implementa actions para aguilizar este proceso, es una simulación básica de clima, humedad (es casi constante) y luminosidad.
